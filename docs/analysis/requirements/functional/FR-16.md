# FR-16: Audit Logging

The system shall immutably log critical user actions (e.g., login, asset creation, permission changes) with relevant details.

**Inputs:** User ID, action type, timestamp, IP address, affected resource ID.
**Processing:** Record audit event in a dedicated, append-only log table.
**Outputs:** Stored audit log entry.
