# FR-07: Define Telemetry Rules

The system shall allow Device Integrators to define threshold-based rules on asset telemetry data.

**Inputs:** Rule expression (e.g., `Boiler_01.Pressure > 120`), associated asset property, threshold value.
**Processing:** Validate rule syntax; store rule definition.
**Outputs:** Stored rule definition.
