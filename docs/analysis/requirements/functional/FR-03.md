# FR-03: Ingest Real-time MQTT

The system shall ingest real-time MQTT messages and parse their payloads.

**Inputs:** Raw MQTT message (topic, payload) from message broker.
**Processing:** Subscribe to configured MQTT topics; receive and buffer messages.
**Outputs:** Buffered MQTT message.
