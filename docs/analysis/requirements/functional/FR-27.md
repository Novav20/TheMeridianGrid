# FR-27: Acknowledge Alerts

The system shall allow Operators to acknowledge active alerts.

**Inputs:** Alert ID, User ID (acknowledging operator).
**Processing:** Update alert status to "Acknowledged"; record timestamp and user.
**Outputs:** Updated alert status.
