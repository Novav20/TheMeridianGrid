# FR-19: Drag-and-Drop Grid

The system shall enable Device Integrators to arrange and resize dashboard widgets using drag-and-drop functionality on a grid.

**Inputs:** Widget ID, new (x, y, width, height) coordinates.
**Processing:** Update dashboard JSON configuration with new widget layout.
**Outputs:** Visually updated dashboard layout.
