# FR-06: Store Asset Metadata

The system shall store arbitrary key-value metadata (e.g., Serial Number, Manufacturer) for each Digital Asset.

**Inputs:** Metadata key-value pairs.
**Processing:** Store in PostgreSQL JSONB column associated with the asset.
**Outputs:** Stored metadata accessible via asset ID.
