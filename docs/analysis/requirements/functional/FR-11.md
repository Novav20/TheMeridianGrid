# FR-11: Data Retention Policies

The system shall automatically enforce configured data retention policies for telemetry data.

**Inputs:** Configured retention period per data type.
**Processing:** Periodically delete or aggregate telemetry data older than the specified retention period.
**Outputs:** Removed/aggregated historical data.
