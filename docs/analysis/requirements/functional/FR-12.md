# FR-12: User CRUD

The system shall allow System Administrators to create, read, update, and disable user accounts.

**Inputs:** User details (email, name, password, etc.), user ID for updates/disables.
**Processing:** Validate inputs; hash passwords; store/retrieve user records.
**Outputs:** New user account, updated user details, disabled user status, user list.
