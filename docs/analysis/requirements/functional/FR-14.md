# FR-14: Secure Authentication

The system shall authenticate users securely using hashed passwords and manage session validity.

**Inputs:** User email, password.
**Processing:** Retrieve user record; hash input password and compare; generate/validate secure session token (HttpOnly cookie).
**Outputs:** Authenticated user session, redirection to dashboard or error message.
