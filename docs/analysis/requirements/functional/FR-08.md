# FR-08: Evaluate Asset Status

The system shall evaluate an asset's operational status (Healthy, Warning, Critical) based on active rules and current telemetry.

**Inputs:** Asset telemetry, active rule definitions.
**Processing:** Continuously evaluate rules against incoming telemetry; determine the most severe status.
**Outputs:** Updated asset status.
