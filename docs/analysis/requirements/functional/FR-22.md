# FR-22: Real-time WebSockets

The system shall push real-time telemetry updates to active dashboard widgets via WebSockets.

**Inputs:** Mapped telemetry data points (from CORE-02).
**Processing:** Backend pushes updates to connected frontend clients; frontend updates relevant widgets.
**Outputs:** Live-updating dashboard widgets.
