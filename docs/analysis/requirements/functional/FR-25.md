# FR-25: Create Alert Instance

The system shall create a new Alert Instance in response to a rule condition breach.

**Inputs:** Rule breach event (asset ID, property, value, timestamp, rule ID).
**Processing:** Create a new record in the Alerts table with "New" status, severity.
**Outputs:** Stored Alert Instance.
