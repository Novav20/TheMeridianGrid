The system shall provide a library of standard visualization widgets including Time-series Line Charts, Radial Gauges, and Numeric Displays.

**Inputs:** Widget type selection.
**Processing:** Instantiate selected widget with default properties.
**Outputs:** New empty widget on the dashboard.
