# FR-09: Asset-level Permissions

The system shall allow System Administrators to assign granular access permissions to users or roles at the asset level.

**Inputs:** User/Role ID, Asset ID, permission type (Read, Write, Execute).
**Processing:** Store permission assignments in a dedicated access control list (ACL) table.
**Outputs:** Stored access permission record.
