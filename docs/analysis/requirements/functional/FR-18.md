# FR-18: Dashboard CRUD

The system shall allow Device Integrators to create, rename, duplicate, and delete dashboards.

**Inputs:** Dashboard name, ID.
**Processing:** Store dashboard configuration (JSON) in the database.
**Outputs:** New dashboard, updated dashboard name, deleted dashboard.
