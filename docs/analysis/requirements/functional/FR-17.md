# FR-17: Developer Mode

The system shall provide a "Developer Mode" that, during initial setup or via configuration, automatically grants full administrative and integration permissions to a single user.

**Inputs:** Environment variable or first-run wizard selection.
**Processing:** Assigns "SuperAdmin" role to the initial user; bypasses certain scope checks.
**Outputs:** User with elevated permissions; visual indicator of Developer Mode.
