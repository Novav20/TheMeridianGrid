# FR-01: Store AAS/DTDL Models

The system shall store digital twin definitions (Asset Administration Shell / DTDL models) in a flexible, schema-less format.

**Inputs:** JSON-formatted DTDL model from Device Integrator.
**Processing:** Validate JSON structure; store in PostgreSQL JSONB column.
**Outputs:** Unique Asset Identity (ID); stored digital twin model.
