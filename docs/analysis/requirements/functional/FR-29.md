# FR-29: Interlock Notifications

The system shall notify System Administrators of any blocked automated control commands due to safety interlock breaches.

**Inputs:** "Interlock Breach" signal.
**Processing:** Create a Critical Alert; send notifications via configured channels.
**Outputs:** Alert notification.
