# FR-21: Data Binding

The system shall allow Device Integrators to bind a widget's data input to a specific property of a Semantic Asset.

**Inputs:** Widget ID, Asset ID, property name.
**Processing:** Store binding configuration within the widget's properties in the dashboard JSON.
**Outputs:** Widget configured to display data from the bound source.
