# FR-10: Asset Lifecycle State

The system shall manage the lifecycle state of a Digital Asset (Draft, Active, Archived).

**Inputs:** Asset ID, desired state (Draft/Active/Archived).
**Processing:** Update asset state; enable/disable telemetry ingestion and rule evaluation based on state.
**Outputs:** Updated asset state.
