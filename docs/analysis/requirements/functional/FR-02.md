# FR-02: Unique Asset Names

The system shall prevent the creation of Digital Assets with non-unique names.

**Inputs:** Proposed asset name.
**Processing:** Check database for existing asset names.
**Outputs:** Error message on duplicate name; successful creation otherwise.
