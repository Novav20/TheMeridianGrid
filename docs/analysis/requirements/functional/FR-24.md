# FR-24: Continuous Rule Evaluation

The system shall continuously evaluate defined rules (e.g., `Asset.Property > Value`) against incoming telemetry data.

**Inputs:** Incoming telemetry data, active rule definitions.
**Processing:** Execute rule expressions; detect condition breaches.
**Outputs:** Rule breach event.
