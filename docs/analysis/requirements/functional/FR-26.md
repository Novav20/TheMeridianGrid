# FR-26: Deduplicate Alerts

The system shall prevent the creation of duplicate active alerts for the same condition on the same asset.

**Inputs:** New rule breach event.
**Processing:** Check for existing "New" or "Acknowledged" alerts for the same condition.
**Outputs:** Update "Last Seen" timestamp of existing alert, or create new alert if unique.
