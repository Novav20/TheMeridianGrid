# FR-23: Historical Data Retrieval

The system shall retrieve and display historical telemetry data for specified time ranges.

**Inputs:** Asset ID, property name, start timestamp, end timestamp.
**Processing:** Query TimescaleDB for data within the range; aggregate if necessary.
**Outputs:** Chart/table display of historical data.
