# Actor: System Administrator

The **System Administrator** manages the platform's infrastructure, security, and user lifecycle.

### Responsibilities
*   User onboarding and account management.
*   Role and permission (RBAC) configuration.
*   Monitoring platform health (CPU, RAM, Disk).
*   Managing data retention policies.

### Primary Goals
*   Maintain system uptime and security.
*   Ensure compliance with operational standards (IEC 62443).
