# Actor: Device Integrator

The **Device Integrator** is responsible for the technical setup and mapping of physical hardware to the digital platform.

### Responsibilities
*   Defining Digital Twin models (AAS/DTDL).
*   Configuring MQTT topic mappings.
*   Building and binding dashboard widgets.
*   Defining operational rules and thresholds.

### Primary Goals
*   Ensure data flows correctly from the machine to the cloud.
*   Create meaningful visualizations for operators.
